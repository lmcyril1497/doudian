# 抖店 - 部署说明

## 如何访问

打开浏览器输入：**https://lmcyril1497.github.io/doudian/**

手机、电脑、任何地方都能打开，不需要任何启动。

## 服务说明

| 服务 | 用途 | 状态 | 需要手动启动？ |
|---|---|---|---|
| GitHub Pages | 托管网页代码 | 永久在线 | ❌ 不需要 |
| Supabase | 云端数据库 | 永久在线 | ❌ 不需要 |

## 常见问题

**Q: 电脑关了还能用吗？**
A: 能。代码在 GitHub，数据在 Supabase，都是云服务器。

**Q: Supabase 会断吗？**
A: 免费版永久运行，不会断。每月 500MB 存储够你用。

**Q: 忘记密码怎么办？**
A: 目前需要联系管理员。后续可加管理员重置功能。

**Q: 怎么给别人用？**
A: 发网址 + 告诉他们用自己手机号登录即可。数据互不影响。
